# FSWD Mini Project

This repository contains the Full Stack Web Development mini project.

## Directory Structure

- `frontend/`: The Next.js full-stack application containing both the React frontend and the backend API routes, alongside Prisma ORM for database connectivity.
- `package.json`: Currently, dependencies are managed within the `frontend/` directory.

To run the application locally:
1. Navigate to the `frontend/` directory: `cd frontend`
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
